Exercício 2: Menu Responsivo

Este exercício demonstra como criar um menu responsivo usando classList.toggle() para mostrar/ocultar o menu em dispositivos móveis.

Redimensione a janela do navegador para menos de 768px de largura para ver o botão hambúrguer. Clique nele para mostrar/ocultar o menu.

Como funciona:

Quando a tela é redimensionada para menos de 768px, o CSS oculta o menu normal e exibe o botão hambúrguer. Ao clicar no botão, o JavaScript adiciona ou remove a classe visible do menu usando classList.toggle().
