Exercício 3: Galeria de Imagens Dinâmica

Este exercício demonstra como usar getAttribute() e setAttribute() para criar uma galeria de imagens dinâmica. Clique nas miniaturas abaixo para alterar a imagem principal.

Paisagem com montanhas e lago

Como funciona:

Quando você clica em uma miniatura, o JavaScript obtém os atributos src e alt da imagem clicada usando getAttribute() e, em seguida, atualiza a imagem principal usando setAttribute().
