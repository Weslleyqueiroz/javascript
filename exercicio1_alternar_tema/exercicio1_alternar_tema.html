Exercício 1: Alternar Tema

Este exercício demonstra como usar classList.toggle() para alternar entre temas claro e escuro.

Clique no botão no canto superior direito para alternar entre os temas.

Como funciona:

Quando você clica no botão, o JavaScript adiciona ou remove a classe dark-theme do elemento body usando o método classList.toggle().

O CSS define estilos diferentes para quando a classe dark-theme está presente, alterando as cores de fundo, texto e outros elementos.

Além disso, o texto do botão é atualizado para refletir a ação que será realizada no próximo clique.
